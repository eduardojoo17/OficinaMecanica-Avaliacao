# Avaliação - Modelagem de Sistemas


### Pedro Pais, Daniel Guimarães, João Eduardo, Alan Silva


##### Oficina Mecânica e Centro Automotivo
- *O Caso de Negócio:*
Sistema Atual: A oficina "AutoFix" utiliza o software "MecânicoSys 2005", instalado apenas no computador do balcão. As ordens de serviço (OS) circulam em pranchetas de papel sujas de graxa.
O Problema Central: O cliente deixa o carro e fica "no escuro". Para saber se o orçamento foi aprovado ou se o carro está pronto, ele precisa ligar. Isso faz com que o mecânico chefe pare o serviço a cada 20 minutos para atender o telefone ou explicar o defeito novamente para a recepção. A aprovação de orçamentos extras demora dias porque depende do cliente ir até lá ou atender o telefone.
- *Personas:*
Carlos (Cliente, 30 anos): Trabalha em escritório. Dor: "Odeio ligar para saber se meu carro está pronto e ficar ouvindo barulho de chave de fenda ao fundo. Queria aprovar o orçamento e ver fotos da peça quebrada pelo celular."
Seu Jorge (Mecânico Chefe, 55 anos): Especialista, mas odeia tecnologia. Dor: "Não consigo trabalhar. A recepção me chama toda hora para explicar a mesma coisa para o cliente no telefone. Quero focar no motor."
